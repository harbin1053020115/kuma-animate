kuma-animate是一个常用动画库；
- 提供了基于animate.css的常见动画类，可以方便的运用到项目中
- 另外，kuma-animate也对以下几个常见组件进行了动画支持：
  - kuma-button
  - alinw/tip
  - alinw/dialog
  - icons
